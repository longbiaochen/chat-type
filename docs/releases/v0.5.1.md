# ChatType v0.5.1

`v0.5.1` is a focused bug-fix release for AI Polish.

## Fixed

- Fixed the ChatGPT-backed AI Polish request body so it matches the current private Codex Responses endpoint requirements.
- Removed unsupported request parameters from the AI Polish call path.
- Preserved safe fallback behavior: if polish fails, ChatType still returns the normalized ASR text.

## Improved

- Settings now accounts for AI Polish attempts, successes, and failures instead of counting only successful provider runs.
- Existing latency history is interpreted so earlier hidden failures show up as failed polish attempts.
