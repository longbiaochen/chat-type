# ChatType v0.1.0

## What It Does

`ChatType` is a lightweight macOS dictation app for a single fast loop:

1. Press `F5`
2. Speak
3. Press `F5` again
4. Get transcribed text pasted into the focused editor, or copied to the clipboard when paste is not safe

`v0.1.0` defaults to the desktop-login route. If Codex is installed on this Mac and signed in with ChatGPT, `ChatType` can transcribe without asking for any API key.

## Quick Start

1. Install and open Codex on this Mac
2. Sign in with your ChatGPT account
3. Open `ChatType.app`
4. Grant microphone permission
5. Grant Accessibility permission if you want auto-paste
6. Focus an editable field and use `F5` to record

## Advanced Recovery

An `OpenAI-Compatible Recovery` route still exists in Settings for troubleshooting and power users. It is not part of the default launch flow.

## Output Path

`v0.1.0` now ships as a single-stage STT product.

- no second AI cleanup pass in the default flow
- OpenAI-compatible recovery uses the official transcription `prompt` parameter
- the desktop-login bridge tries the same prompt and falls back automatically if that private path rejects it
- optional hidden `transcription.hintTerms` preserve critical terms without another model call

## Known Limitations

- `ChatType` depends on a local signed-in Codex desktop session in the default path
- the backend path is private and may change upstream
- the release build is ad-hoc signed and not notarized yet
- the desktop bridge prompt path is best-effort and may need to fall back to plain transcription

## Performance Notes

- auth warm-up now caches the desktop login state for repeated dictations
- per-run latency is written to `~/Library/Application Support/ChatType/latency.jsonl`
- benchmark the packaged path with `./scripts/benchmark_stt.sh <3s.wav> <10s.wav> <30s.wav>`

## Install Notes

Release assets for `v0.1.0` are packaged as:

- `ChatType-0.1.0-macos-arm64.zip`
- `ChatType-0.1.0-macos-arm64.dmg`

If macOS blocks the app:

- right-click `ChatType.app` and choose `Open`
- or run:

```bash
xattr -dr com.apple.quarantine /path/to/ChatType.app
```
