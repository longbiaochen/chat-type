# VoiceDex v0.1.0

## What It Does

VoiceDex is a lightweight macOS dictation app for a single fast loop:

1. Press `F5`
2. Speak
3. Press `F5` again
4. Get cleaned-up text pasted into the focused editor, or copied to the clipboard when paste is not safe

`v0.1.0` defaults to the stable OpenAI-compatible path for transcription and AI cleanup. The older Codex bridge remains available, but it is now explicitly experimental.

## Quick Start

1. Export your key before launching the app:

```bash
export OPENAI_API_KEY=your_key_here
```

2. Open `VoiceDex.app`
3. Grant Microphone permission
4. Grant Accessibility permission if you want auto-paste
5. Focus an editable field and use `F5` to record

## Install Notes

This build is ad-hoc signed and not notarized yet.

If macOS blocks the app:

- right-click `VoiceDex.app` and choose `Open`
- or run:

```bash
xattr -dr com.apple.quarantine /path/to/VoiceDex.app
```

## Known Limitations

- `OPENAI_API_KEY` must be provided through the environment that launches the app
- the release build is not notarized yet
- the app is currently optimized for Apple Silicon macOS workflows
- screenshots and guided first-run onboarding are still pending

## Experimental Bridge Disclaimer

`codexChatGPTBridge` is still included for experimentation with local Codex login state, but it is not the recommended or supported default for public use in `v0.1.0`.
