# ChatType v0.5.0

`v0.5.0` adds long-form text polish after ASR while keeping ChatType's conservative paste behavior unchanged.

## Highlights

- Hybrid pipeline: ASR -> terminology normalization -> optional AI polish -> protective normalization -> paste/clipboard.
- Text polish uses ChatType-managed ChatGPT Auth only, with no separate polish API key.
- The ChatGPT backend Responses endpoint and model are visible in Settings for inspection and recovery.
- Settings now uses a workflow sidebar for Account, Dictation, AI Polish, Terminology, Paste, and Advanced configuration.
- Long Chinese or mixed-language dictation can be rewritten into concise, agent-friendly plan bullets.
- Terminology matching now has higher recall for accented Latin ASR drift while preserving technical literals.
